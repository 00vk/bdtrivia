# MVP — «Kahoot у нас дома»

Браузерная викторина-мультиплеер. Одна страница, два режима: хост (`index.html`) и игрок (`index.html?room=CODE`).

**Стек:** GitHub Pages (статика) · Firebase RTDB (Spark) · Vanilla JS · qrcode.js
